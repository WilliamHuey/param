# Tower Param Validator

## Installation

node.js:

```bash
$ npm install tower-param-validator
```

browser:

```bash
$ component install tower-param-validator
```

## Example

```js
route.use(require('tower-param-validator'));
```

Or, lazy load it? (todo)

```js
route.load('validate', 'tower-param-validator');
```
